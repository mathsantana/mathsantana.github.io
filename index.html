Fulano de Sal

Aplicativo de gerenciamento de pedidos da marmitaria Fulano de Sal.

Tecnologias

O aplicativo Android foi feito com framework React Native, servido por um backend, feito com o framework Spring Boot. Para armazenar os dados, foi utilizado um banco de dados relacional PostgreSQL.

Desafio Codex

Desafio realizado no processo seletivo da empresa júnior de Ciência da Computação, Codex.
